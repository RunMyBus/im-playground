# Claude Configuration for Igniting Minds Website

## Project Overview
This is the Igniting Minds (IM) website built with Next.js, featuring Material-UI components and various interactive features.

## Tech Stack
- **Framework**: Next.js 14.0.4
- **UI Library**: Material-UI (MUI) v5.15.2
- **Maps**: React Google Maps API, Mapbox GL
- **Charts**: MUI X Charts
- **Forms**: Formik
- **Animations**: AOS (Animate On Scroll)
- **Icons**: React Icons, MUI Icons

## Development Commands
- `npm run dev` - Start development server on port 3000
- `npm run build` - Build production version
- `npm run start` - Start production server on port 3000
- `npm run lint` - Run ESLint

## Key Dependencies
- React 18 with Next.js
- Material-UI ecosystem (@mui/material, @mui/x-charts, @mui/x-data-grid)
- Form handling with Formik
- Map integration with Google Maps and Mapbox
- Image processing with Sharp
- Animation libraries (AOS, Typed.js)

## Project Structure
- `src/` - Source code directory
- `public/` - Static assets
- `server.js` - Custom server configuration
- `next.config.js` - Next.js configuration

## Environment
Requires `.env.local` for environment variables (API keys, etc.)
